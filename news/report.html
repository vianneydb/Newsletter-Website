Assignment Report

May 2, 2024 by Vianney Diaz

Major assignemnt report, this is how I was inspired.

My project is about a website newspaper page, I chose it because in high school I took some classes in web programming and I made many pages with different styles and purposes so, I just implemented some ideas from my past projects.

When I read the assignment sheet, I was a little bit lost and I didn't know what to do, at first I thought about a hand-made product, like cookies or a painting, but was a little bit difficult for me to implement the other four projects, so then I think about some way to publish my works, then the idea of a website newspaper page came up to my mind, where I published my works as a news and everyone can visit it.

I used the IDE Visual Studio Code to program my website and I used HTML5 and CSS language programming, also the library Bootstrap helped me a little bit to make it responsive. Finally, I used Infinity Free to host my website and publish it. You can access it from any device, but I don't have an SSL certificate so, some devices will can't access it because of the secure restrictions of the device.

The total process of the website page took me around 1 week because I didn't remember a few things about web programming and I needed to re-learn, so I saw some YouTube videos and then started to work
